First ensure that pip is installed and use this opensource library:
pip install pyautogui keyboard


after installed and the other file is compiled, run the following:
python auto_clicker.py

Enable Clicking:

Press the F8 key. The script will start simulating 100 mouse clicks each time you click the mouse button.
Disable Clicking:

Press the F8 key again to stop the automatic clicking.
Exit the Script:

Press the Esc key to terminate the script.
